# Технічне завдання — NoiseMap Aklima
## Веб-застосунок «Акустичний калькулятор зовнішніх блоків на карті»
**Версія:** 1.0 · **Дата:** 09.07.2026 · **Замовник:** Aklima

---

## 1. Мета продукту

Веб-застосунок для інженерів та менеджерів Aklima:
1. Знаходить об'єкт на карті (адреса або координати).
2. Розміщує зовнішні блоки (тепловий насос, ККБ, чилер, VRF) на землі/стіні/даху з висотою монтажу.
3. Автоматично підтягує контури та висоти будівель з OpenStreetMap (Overpass API).
4. Розраховує рівень звукового тиску (дБА) за **ISO 9613-2** у точках-приймачах на фасадах.
5. Будує шумову карту (ізолінії) та генерує PDF-звіт.

**НЕ використовувати Google Maps** — ліцензія забороняє витягувати геометрію будівель. Тільки OpenStreetMap (ODbL).

## 2. Архітектура

**MVP: увесь розрахунок на клієнті** (ISO 9613-2 — прості формули, сервер не потрібен).

```
[Браузер: React + MapLibre GL JS]
├── Overpass API → будівлі OSM (кешувати!)
├── Photon API → геокодинг (photon.komoot.io)
├── turf.js → геометрія, ізолінії
├── Web Worker → ISO 9613-2 розрахунок
└── [Supabase] → проєкти, каталог, users
    └── Edge Function або pdf-lib → PDF-звіт
```

**Стек:**
- Frontend: React 18 + TypeScript + Vite + Zustand
- Карта: MapLibre GL JS + OpenFreeMap тайли (без ключа)
- Backend: Supabase (free tier, Postgres + PostGIS)
- PDF: pdf-lib + html-to-image (клієнтська генерація)
- Хостинг: Vercel
- CI/CD: GitHub Actions

## 3. Функціональні вимоги

### 3.1. Карта
- Базова карта: MapLibre GL JS + OpenFreeMap векторні тайли
- Пошук адреси через Photon API, введення координат вручну
- Шари: будівлі (з висотами), джерела шуму, приймачі, екрани, ізолінії шуму, легенда дБА
- Вимірювання відстаней лінійкою
- Радіус роботи: ~300 м від джерела

### 3.2. Будівлі (Overpass API)
- Автозавантаження полігонів у радіусі 150 м (налаштовується)
- Висота: тег `height`, або `building:levels × 3 м`, або дефолт 6 м
- Ручне малювання полігону для нових будівель (яких немає в OSM)
- Маркування: «захищена» (житло, ставляться приймачі) / «нейтральна» (гараж)

### 3.3. Джерела шуму
- Каталог обладнання в JSON (brand, model, режими, Lw октавні 63-8000 Гц)
- Якщо немає октавних — сумарний Lw(A) + типовий спектр компресорно-вентиляторного агрегату
- Параметри джерела:
  - висота `hs` (м)
  - тип монтажу → DI поправка:
    - вільно (Q=2, +3 дБ)
    - біля стіни (Q=4, +6 дБ)  
    - у куті (Q=8, +9 дБ)
  - кількість однакових блоків (енерг. сумування: +10·log₁₀(n))
  - режим день/ніч (різні Lw)

### 3.4. Приймачі («вікна»)
- Автогенерація: 1 м перед фасадами захищених будівель, крок 3-5 м, висоти 1,5 м та 4,5 м
- Ручне додавання у довільну точку
- Виділення найгіршого приймача (макс. дБА)

### 3.5. Екрани
- Малювання лінією з висотою
- Будівлі — автоматично як екрани (дифракція через верхнє ребро)
- MVP: тільки верхнє ребро (ISO 9613-2, п.7.4)

### 3.6. Розрахунковий рушій ISO 9613-2

Для кожної пари джерело→приймач, по октавах 63-8000 Гц:
```
Lp(f) = Lw(f) + DI − Adiv − Aatm(f) − Agr(f) − Abar(f)
```

- **Adiv** = 20·log₁₀(d) + 11 (геометрична розбіжність)
- **Aatm** = таблиці ISO 9613-1 при 10°C / 70% RH
- **Agr** = альтернативний метод п.7.3.2, G=0.5 (G=0 асфальт, G=1 газон)
- **Abar** = Dz = 10·log₁₀(3 + (C₂/λ)·δ), max 20 дБ; δ = різниця ходу через ребро
- **DI** = 10·log₁₀(Q) від типу монтажу
- Ray casting 3D для перевірки прямої видимості (2D перетин + порівняння висот)
- Сумування джерел: Lp,total = 10·log₁₀(Σ 10^(Lp,i/10))
- А-зважування: стандартні поправки по смугах → дБА

### 3.7. Результати
- Таблиця: будівля, відстань, дБА день/ніч, норма, запас, статус (🟢🟡🔴)
- Шумова карта: сітка 2×2 м @ 1.5 м висота → ізолінії 35/40/45/50/55/60 дБА
- Нормативи:
  - Польща: житло 50/40 дБА (день/ніч), садиби 55/45
  - Україна: житло 55/45 дБА (день/ніч)
- PDF-звіт: скриншот карти + вхідні дані + таблиця + методика ISO 9613-2 + дисклеймер ±3 дБ + логотип
- Збереження проєкту (Supabase + share link)

### 3.8. Нефункціональні
- Розрахунок 5 джерел × 200 приймачів × 8 октав < 1 с (Web Worker)
- Шумова сітка < 10 с
- Мови: PL / UA / EN (i18n)
- Mobile-friendly перегляд, desktop редагування

- Generic Inverter Split 3.5 кВт: Lw=58 дБА
- Для кожної: октавні Lw або сумарний + тип спектру
